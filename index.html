Portable Stdlib

  • Flax's stdlib is written entirely in itself
  • Compiled Flax programs only need the libc to run

Language Goals

Flax is a systems programming language. Not for backend "systems", but for real, low-level systems. Its main target is the OSDev community, positioned as a replacement for traditional languages that monopolise this field, like C and C++.

Naturally, a few features distinguish it from C++ and various other high-level languages:

  •   Clean syntax without thrown-in afterthoughts
  •   High expressibility while still maintaining raw control
  •   Ease of portability, especially the standard library
  •   LLVM backend ensures Flax can target nearly any platform
